## How to (Build from source)

**Requirements**
- JDK 17
- Maven

**Steps**
1. Clone this repository:
    ```
    git clone https://github.com/TheSpeedCubing/cpu-scheduling-visualizer.git
    cd cpu-scheduling-visualizer
    ```
2. Run the build script:
    ```
    ./build.sh
    ```

3. After building, you'll find the server JAR located at:
    ```
    ./target/cpuschedulingvisualizer-R0.1-SNAPSHOT.jar
    ```
